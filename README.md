# TODOLIST

Essa é uma simples aplicação de lista de afazeres.

# Tecnlogias

## Frontend

Foi utilizado o framework Angular junto com o boostrap.

## Backend

Foi utilizado o framework Grails.

## Banco de Dados

Foi utilizado o postgresql, porém, pode ser mudado sem problemas.
